### Githug 安装及命令

1. [Github链接]: https://github.com/Gazler/githug

2. 安装Ruby 1.8.7 or higher

3. gem install githug

4. 如果遇到权限问题，请加上`sudo`：sudo gem install githug

4. Commands
   - play 
   - hint 
   - reset 
   - levels
